# md5Creator With Py3

use `generateMD5` function to make string to hash md5 

# Link

```
https://github.com/Zrexer/md5Creator
```

# Work: Method1
```
git clone https://github.com/Zrexer/md5Creator
cd ./md5Creator
pip install -r requiremets.txt
python3 md5Creator.py
```
